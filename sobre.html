Sobre mim

Meu nome é André Protázio, sou publicitário formado pelo Instituto Federal do Amazonas (IFAM). Durante a graduação, desenvolvi habilidades na produção visual de vídeos, o que acabou me direcionando para o design. Em conjunto com a equipe, participei do desenvolvimento de embalagens com foco na experiência do consumidor.

Meu TCC, elaborado em parceria, abordou a implementação de uma nova identidade visual para um jornal local em uma interface web mobile. Desde então, tenho me especializado na criação de interfaces e no desenvolvimento de páginas web, com ênfase em acessibilidade e usabilidade.

Estou aprimorando meu Front-End com HTML e CSS pela Alura, buscando habilidades em React e Next.js para desenvolver códigos colaborativos e facilitar futuros ajustes por outros desenvolvedores.

Habilidades

  • HTML 5 / CSS
  • UI / UX design
  • Edição de vídeos
  • Motion design
